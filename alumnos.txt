Tema: Sistema de gestión de obras urbanas con manejo de POO, importación de datasets desde
un archivo csv y persistencia de objetos con ORM Peewee en una base de datos SQLite.

Se requiere desarrollar un software en Python para gestionar las obras urbanas de la Ciudad de
Buenos Aires, tomando como origen de datos un dataset público del gobierno de la ciudad y
haciendo uso del modelo ORM de la librería ó módulo “peewee”. Para el manejo y operaciones
con el dataset se debe utilizar la librería ó módulo “pandas”. Y para el manejo y operaciones con
arrays pueden utilizar la librería ó módulo “numpy”.
